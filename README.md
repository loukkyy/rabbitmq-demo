## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

* npm
  ```sh
  npm install npm@latest -g
  ```
* Rabbit MQ (TODO)

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/loukkyy/rabbitmq-demo.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
